# \# Kerala Assembly Election Analysis (2011–2021)



This project analyses Kerala Legislative Assembly elections from 2011 to 2021 using data science and machine learning techniques. It explores vote share trends, seat margins, alliance dynamics, and turnout patterns across constituencies.



This project aim to generate political insights using real election data and apply statistical, visual, and machine learning tools as progress.



\## Problem statement

* How have vote shares changed across elections?
* Which alliances gained or lost ground between 2011, 2016, and 2021?
* What role do incumbency and turnout play in outcomes?
* Can we build a model to predict who might win a seat based on past patterns?



\## Tools used

* Python
* Jupyter notebook
* pandas, numpy, matplotlib, seaborn
* Git and GitHub
